# Zwitter

Zwitter is a real time chat app based on a python module [NiceGui](https://nicegui.io/) supporting auto avatar genration and dark and light theme. 

## Requirements

- NiceGui
- MiddleWare
- FastApi

Note: Python 3.10+ Recommended

## How to run?
1.) ``` pip install -r requirements.txt ```

2.) ``` python main.py ```
